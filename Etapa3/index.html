Biblioteca do OpenCV

Nesta etapa é feita uma breve explicação da bilioteca OpenCV que foi utilizado neste projeto, bem como algumas das funções utilizadas

OpenCV

OpenCV é uma biblioteca open-source e extensivamente utilizada na área de visão computacional. Desenvolvida para ser eficiente computacionalmente e voltada para aplicações em tempo real.

A biblioteca contém mais de 500 funções, dentre as quais são aplicadas para manipulação de imagens e vídeos. Entre suas funcionalidades estão aplicação de filtros, leitura e gravação de imagens e vídeos, mudanças de escala e dimensionalidade, extração de contornos, entre outros.

Funções do OpenCV

cv.HoughCircles()

Essa função utiliza a transformada de Hough para detectar círculos dentro da imagem

cv2.cvtColor()

cv2.cvtColor converte a imagem de um sistema de cores para outro como RBG para tons de cinza(GRAY) ou RBG para HSV

O sistema HSV permite ver melhor o contraste da imagem, identificando melhor as cores presentes

Um exemplo de uso

cv2.findContours()

Com o cv2.findCountours é possível, a partir de uma imagem binária, encontrar contornos, identificados pela variação de tonalidade

Após encontrados os contornos a própria biblioteca possui funções para calcular parâmetros como área, perimetro, entre outros.
