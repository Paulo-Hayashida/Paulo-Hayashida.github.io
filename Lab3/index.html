Introdução

Neste laboratório foi possível construir e visualizar as propriedades das cameras estereos, onde através de imagens 2D é possível trazer uma sensação de profundidade, utilizando lentes de cores diferentes

Parte 1

Para a primeira etapa foram disponibilizados 2 links referentes a documentações contendo excplicações teóricas sobre geometria epipolar e confecção de cameras 3D .

Link 1: https://learnopencv.com/introduction-to-epipolar-geometry-and-stereo-vision/

Ao converter um ponto 3D em sua representação num espaço 2D, também chamada de projeção planar, perdemos a informação de profundidade da imagem. E para recuperar essa informação, não é possível apenas com uma imagem. Mas ao possuir duas imagens e tendo bem definidas suas posições os pontos projetados de um dado ponto 3D, é possível recuperar esse parâmetro, através de triangulação.

Disparidade Stereo é obtida quando se considera que as imagens estão paralelas e então a diferença entre os pontos y é 0.

Link 2: https://learnopencv.com/making-a-low-cost-stereo-camera-using-opencv/

Esse link nós ensina como montar e calibrar uma câmera estereoscopica utilizando códigos em python e imagens das câmeras do lado direito e do lado esquerdo

Os óculos 3D funcionam, pois cada olho enxerga uma imagem diferente. A maior diferença está na posição horizontal, essa diferença é chamada de disparidade horizontal e ela é maior quanto mais próximo um objeto se encontra dos nossos olhos.

Assim utilizando duas imagens diferentes e filtros na cor vermelho e ciano, pode-se criar uma ilusão de profundidade nos filmes 3D. Ao gerar esse efeito estereoscopico com filtros utilizamos o método chamado de 3D anáglifo.

Parte 2

Nesta etapa foi construido uma câmera estereosccopica utilizando duas câmeras, uma base e fita adesiva disponibilizada pelo professor

Parte 3

(A)

A partir do diretório do github, foram baixados os três arquivos calibrate.py, movie3d.py e capture_images.py

(B)

Para realizar a calibração do exemplo executado, deve-se realizar a calibração como com uma única câmera, porém para cada uma das câmeras. Obtendo os parâmetros intrínsecos de cada câmera

Em seguida é necessário encontrar a matriz que relaciona as duas câmeras, com essa matriz é possível executar a retificação

Na retificação são necessários as matrizes intrínsecas e coeficientes de distorção de cada uma das câmeras. Como resultado é retornada a projeção da imagem capturada por cada câmera, sendo que as duas projeções se encontram no mesmo plano

(C)

No arquivo de parâmetros de calibração (xml) foram salvos os mapas de correção e de retificação de cada câmera. Quando essas duas matrizes são aplicadas a uma imagem original, elas são convertidas para produzir as imagens de cada câmera projetadas num mesmo plano
