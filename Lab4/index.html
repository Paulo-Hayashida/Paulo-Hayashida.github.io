Introdução

Neste laboratório foi possível construir e visualizar as propriedades das cameras estereos, onde através de imagens 2D é possível trazer uma sensação de profundidade, utilizando lentes de cores diferentes

Parte 1

Para a primeira etapa foram disponibilizados 2 links referentes a documentações contendo excplicações teóricas sobre geometria epipolar e mapas de disparidade para estimação de profundidade.

Link 1: https://learnopencv.com/introduction-to-epipolar-geometry-and-stereovision/.

O primeiro link assim como visto no Laborátorio 3 de camera estero, nos ensina que a partir duas imagens e tendo bem definidas suas posições os pontos projetados de um dado ponto 3D, é possível recuperar esse parâmetro de profundidade através de triangulação.

Caso a diferença entre os pontos no eixo y (altura) seja 0, então estamos trabalhando com a disparidade Stereo.

Link 2: https://learnopencv.com/depth-perception-using-stereo-camera-python-c/

Parte 2

Nesta etapa a partir da câmera construida no Laboratório 3 de câmera estereo, executou-se os passos a seguir

Parte 3

(A)

Executou-se a calibração, como no Laboratório 3, com o programa calibrate.py e utilizando 24 novas imagens para melhorar os a precisão dos próximos passos

Após a execução é gerado o arquivo params_py.xml, que possui os parâmetros para associar uma câmera da outra.

(B)

Adaptou-se código para a entrada do arquivo params_py.xml e executou-se o código disparity_params_gui.py.”

Ao executar o programa é possível ajustar os parâmetros do casamento de blocos através de uma janela, podendo ajustar configurações dos filtros, tamanho do bloco e núemro de disparidades.

Ajustando os parâmetros para identificar com certa precisão espacial o objeto, no caso foi utilizado um caderno, e ao final encerrando o programa com a tecla "Esc" é gerado o arquivo depth_estmation_params_py.xml, que será utilizado no próximo passo para calculo de profundidade.
